# My personal Portfolio Website

A modern, responsive personal portfolio for DevOps engineers built with HTML, CSS, and JavaScript.

## Features
- Professional, technical, and minimal design
- Light/dark theme toggle (button always visible)
- Terminal-inspired header and code flow animations
- Sections: About, Experience, Skills, Contact
- Contact section with LinkedIn, Gmail, and GitHub icons and addresses
- Responsive layout for desktop and mobile

## Setup
1. Open `index.html` in your browser to view the website.
2. Customize content in `index.html`, `style.css`, and `script.js` as needed.

## Customization
- Update your name, contact info, experience, and skills in `index.html`.
- Modify styles in `style.css` for your preferred look.
- Enhance interactivity or animations in `script.js`.
